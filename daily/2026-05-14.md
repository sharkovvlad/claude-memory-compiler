### Session — Streak 7 days: verdict on current mp4 + improved prompt template + KB checklist for prompt-writing agents

**Контекст:** пользователь получил от Nano Banana 6-секундное видео для сценария «Стрик 7 дней» (Номс с пиксельными очками и пламенем). Спросил подходит ли + как улучшить промпт.

**Анализ текущего mp4:** NOT suitable. Проблемы:

1. **Перспективная камера с плиточным полом** — пол semi-transparent (alpha-blended с magenta) → создаст pink halo вниз, который **математически unfixable** (см. KB 7.5). Та же проблема что была со Streak 3 days "splash в желейную лужу".
2. **Танец / горизонтальное движение** — direction-specific motion → невозможен ping-pong loop (KB 7.6).
3. **Reflections под Номсом** — alpha-blending, дублирует pink halo.
4. Pillarboxes стандартные 1280×720 — uncritical, auto-detect разберётся.

**Вывод:** ни одна секунда видео не годится. Нужна перегенерация.

**Композировал новый промпт** для Nano Banana image-to-video:

- 6 sec duration (Nano Banana minimum)
- Seamless loop (final frame = first)
- Locked camera, orthographic front-facing
- **Periodic motions only**: idle vertical sine bounce, flame pulsation (8-bit), sunglasses glow flicker, binary code Matrix stream
- **Hands stay raised** (V-pose), no waving/rotation
- Explicit negatives: no floor, no reflective tiles, no dancing, no walking, no perspective, no watermark, no scene transitions

**KB updates (для будущих агентов-промптеров):**

1. **Обновлены секции 8.3 и 8.4** (image-prompt и video-prompt templates) — добавлены критические негативы:
   - No reflective floor tiles, no glossy ground surface (новый акцент после Streak 7)
   - No directional movement, no perspective view (новый блок)
   - Acceptable motions are PURELY PERIODIC (explicit list)

2. **Новая секция 8.4b «Чеклист для агентов, которые ПИШУТ промпты»** — explicit checklist с MUST positives + MUST NOT negatives + таблицей распространённых ошибок промпт-агентов (с указанием в какой секции KB искать lesson).

**Lessons для других агентов в проекте:**

- Когда генеришь image-to-video промпт для NOMS-стикера — **обязательно прогони его через checklist 8.4b** перед отправкой пользователю. Хорошие image-промпты ≠ хорошие video-промпты: image может быть стационарный, video должен быть loopable и без environmental interaction.
- Промпт-агент должен **знать KB sections 7.5 (alpha-blending), 7.6 (ping-pong), 8.4b (checklist)** — без этого выдаст промпт что приведёт к unfixable pink halo или to direction-specific motion.
